# 📊 Sales & Insights Dashboard (Microsoft Excel)

## 📌 Project Overview
This project is an interactive **Sales & Insights Dashboard** created using Microsoft Excel. It analyzes sales performance, delivery time, product contribution, city-wise order value, customer membership, and courier performance to provide meaningful business insights.

## 📈 Dashboard Highlights

The dashboard includes:

- Warehouse-wise Average Delivery Time
- Product Category Contribution to Total Order Value
- City-wise Contribution to Total Order Value
- Average Delivery Time Analysis
- Regular vs Premium Member Contribution
- Courier Partner Performance Analysis

---

## 📊 Key Insights

### 🚚 Warehouse Performance
- Warehouse W03 has the highest average delivery time.
- Warehouse W04 delivers orders the fastest.

### 🛍️ Product Contribution
- Product categories contribute almost equally to total order value.
- No single category dominates overall sales.

### 🌍 City-wise Sales
- Bengaluru and Pune contribute the highest order values.
- Patna has the lowest contribution.

### 👥 Membership Analysis
- Premium Members contribute approximately **53%** of total order value.
- Regular Members contribute approximately **47%**.

### 🚛 Courier Performance
- Delhivery provides the fastest average deliveries.
- Ecom Express has the highest average delivery time.
